🌙 The clone kit

One source of truth for how the Night Trail looks, speaks, and behaves. Four files. Clone them — never re-invent them. Source of truth: ydc-canvas/styles.css.

1The four files

📋 tokens.css

Every custom property, both themes. <link> it or copy the block. Changes land here first, then everywhere.

📦 tokens.json

The same tokens for non-CSS consumers — native shells, scripts, decks. Includes the type scale and the brand laws.

🧹 lint.js

Tokens-as-lint. Zero-dependency Node script — point it at any .css / .html / .js file and it reports drift from the tokens and the laws.

📖 index.html

This page — the human-readable reference for everything above. View source freely; it follows its own laws.

4Shape — soft corners, pill actions

--r-md: 16px cards and inputs · --r-lg: 22px heroes and panels · 999px pills and chips. Nothing on the trail has a sharp corner — the world is friendly before it is impressive.

5Voice — the dignity guard

tier stands aloneA mastery tier never appears beside an age or a score. Stages describe the road, never the person.
no emoji on struggleRevision and retry states speak plainly and warmly. Celebration gets the confetti; struggle gets respect.
OOS voiceNever "dropout", never remedial. Out-of-school youth are earners-in-waiting; the road meets them where they are.
honest framesAnything designed-but-not-wired says so, out loud, in the UI. Nothing pretends.
MM by CCO onlyEvery Myanmar string ships with a CCO signature. The lint (R4) flags any glyph that appears without one.

6The fences — non-negotiables

3G lawZero images, zero webfonts, zero external asset calls on the learning trail. (This page follows it too — view source.)
minor-data P0All demo learner data is invented (seeded PRNG 20260719). No real minor's data, ever, in any demo.
publish lawEvery deploy: byte-diff + served checks, and every gate event disclosed.
consent before dataHeavy media loads only after a tap, with its MB cost stated first — the video lane is the reference pattern.

7Tokens-as-lint — how to run it

node lint.js path/to/file.css path/to/page.html
R1-colorHardcoded hex outside the palette. Fix: use the token, or propose the color as a new token.
R2-typefont-family outside var(--font). (First live run suggests we need a --font-mono token in v1.1 — see mirror below.)
R3-3G<img>, @font-face, remote url() or <link> — the 3G law, enforced mechanically.
R4-MMAny Myanmar glyph — ships only with CCO sign-off.

The honest mirror — lint vs ydc-canvas itself, 2026-07-24

40 findings on the source property: 34× R1 (status greens, gradient stops, one #000), 5× R2 (mono stacks — v1.1 token candidate), 1× R1 in lesson.css. index.html: clean. The lint is a mirror, not a verdict — findings get fixed or recorded as signed exceptions. The exception register opens with the CTO.
